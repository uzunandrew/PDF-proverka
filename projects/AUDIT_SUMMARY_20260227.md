## 3. Системные проблемы (повторяются в нескольких проектах)

### 3.1. Нормативные ссылки
- **ЭГ, ЭМ2, ЭР:** Неполные ведомости ссылочных документов, отсутствуют ссылки на актуальные СП 484/485/486.1311500.2020
- **ЭГ:** РД 34.21.122-87 без пометки о добровольном статусе

### 3.2. Система заземления
- **ЭГ:** Противоречие по шпунтовым заземлителям (изм.1 исключило, но на чертежах остались)
- **ЭС:** Не указан тип системы заземления
- **ЭР:** Не указан тип системы заземления, нет точки разделения PEN

### 3.3. ДСУП / уравнивание потенциалов
- **ЭГ:** Разводка ДСУП передана собственникам квартир
- **ЭМ2:** ДСУП в ванных не описана
- **ЭС:** Отсутствуют сведения о ОСУП/ДСУП

### 3.4. Кабельная продукция и ОКЛ
- **ЭМ1:** ОКЛ лотки без пожарного сертификата, EI60 вместо EI90
- **ЭМ2:** Не указан класс огнестойкости ОКЛ (E30/E60/E90)
- **ЭР:** Кабели собственных нужд ВВГнг-LS — проверить необходимость FR для СПЗ
- **Фасад:** Не указан полный индекс пожарной безопасности по ГОСТ 31565

### 3.5. Отсутствие расчётов
- **ЭС:** Отсутствует расчёт заземляющего устройства
- **ЭР:** Отсутствует расчёт электрических нагрузок
- **Фасад:** Отсутствует расчёт нагрузок и падения напряжения

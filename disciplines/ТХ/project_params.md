## Что искать в тексте (дисциплина ТХ)
- Нормативные ссылки (СП 118, СП 113, СП 154, СанПиН и др.)
- Назначение встроенных помещений (торговля, общепит, офисы, медицина)
- Количество и площади встроенных помещений
- Характеристики технологического оборудования (мощность, производительность)
- Количество машиномест в паркинге, размеры
- Расчётные нагрузки на перекрытия от оборудования
- Расходы воды, стоков, электроэнергии от технологии
- Тепло- и холодовыделения от оборудования
- Требования к мусороудалению

```json
{
  "building_type": "МКД со встроенными",
  "commercial_area_m2": 0,
  "commercial_functions": [],
  "parking_type": "",
  "parking_spaces_total": 0,
  "parking_spaces_mgn": 0,
  "parking_levels": 0,
  "food_service_seats": 0,
  "food_service_type": "",
  "medical_rooms_count": 0,
  "retail_area_m2": 0,
  "office_area_m2": 0,
  "waste_system": "",
  "loading_dock_count": 0,
  "equipment_total_power_kW": 0,
  "equipment_water_demand_m3h": 0
}
```

## Описание полей
| Поле | Описание | Пример |
|---|---|---|
| building_type | Тип здания | "МКД со встроенными" |
| commercial_area_m2 | Общая площадь коммерческих помещений | 2500 |
| commercial_functions | Функции встроенных помещений | ["торговля", "общепит", "офисы"] |
| parking_type | Тип паркинга | "подземный 2 уровня" |
| parking_spaces_total | Всего машиномест | 120 |
| parking_spaces_mgn | Машиномест для МГН | 12 |
| parking_levels | Количество уровней паркинга | 2 |
| food_service_seats | Количество посадочных мест (общепит) | 50 |
| food_service_type | Тип предприятия питания | "ресторан", "кафе", "столовая" |
| medical_rooms_count | Количество медицинских кабинетов | 3 |
| retail_area_m2 | Площадь торговых помещений | 800 |
| office_area_m2 | Площадь офисных помещений | 500 |
| waste_system | Система мусороудаления | "мусоропровод + мусорокамера" |
| loading_dock_count | Количество загрузочных | 2 |
| equipment_total_power_kW | Суммарная мощность оборудования, кВт | 150 |
| equipment_water_demand_m3h | Расход воды на технологию, м3/ч | 2.5 |

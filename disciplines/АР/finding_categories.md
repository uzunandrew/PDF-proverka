| category | Описание |

| `facade` | Фасадные решения: отделка, витражи, входные группы, ограждения |
| `roof` | Кровля: уклоны, водоотведение, парапеты, примыкания |
| `insulation` | Теплозащита: конструкция стен, теплотехнический расчёт, мостики холода |

| `sound` | Звукоизоляция: индекс Rw стен и перекрытий, ударный шум |
| `documentation` | Документация: несоответствие между чертежами, спецификациями, ведомостями |

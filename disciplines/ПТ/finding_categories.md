Категории замечаний для дисциплины ВПВ/АУПТ:

| category | Описание |
|---|---|
| `fire_hydrants` | Пожарные краны ВПВ: размещение, комплектация, радиус действия, напор |
| `sprinkler` | Спринклерные системы: оросители, расстановка, интенсивность, узлы управления |
| `drencher` | Дренчерные системы: завесы, расход, побудительные системы |
| `fire_pumps` | Пожарные насосные: основной/резервный насос, жокей-насос, автоматика, обвязка |
| `reservoir` | Пожарные резервуары: объём, подключение, неприкосновенный запас |
| `pipe_network` | Трубопроводные сети: диаметры, кольцевание, материал труб, арматура |
| `hydraulic` | Гидравлика: расчёт расходов, напоров, потерь давления, скоростей |
| `automation` | Автоматика и управление: пуск насосов, сигнализация, связь с АПС |
| `documentation` | Соответствие чертежей спецификации, оформление, комплектность |
| `normative_refs` | Устаревшие или неверные ссылки на нормативные документы |

Формат в JSON:
```
"category": "fire_hydrants|sprinkler|drencher|fire_pumps|reservoir|pipe_network|hydraulic|automation|documentation|normative_refs"
```

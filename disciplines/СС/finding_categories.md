Категории замечаний для дисциплины СС (Слаботочные системы):

| category | Описание |
|---|---|
| `scs` | Структурированная кабельная система: категория кабеля, коммутация, серверная, длины линий |
| `cctv` | Видеонаблюдение: камеры, зоны покрытия, хранение, питание |
| `access_control` | СКУД: считыватели, контроллеры, замки, интеграция |
| `intercom` | Домофон/видеодомофон: панели, мониторы, замки, ёмкость |
| `fire_alarm` | ОПС/АУПС: извещатели, шлейфы, ПКП, интеграция с СОУЭ и инженерными системами |
| `pa_system` | СОУЭ: тип системы, зоны оповещения, мощность, кабели |
| `tv_radio` | ТВ/радио: антенное хозяйство, усилители, разводка |
| `dispatch` | Диспетчеризация: лифты, ИТП, пожарная, водомерный, АСКУЭ |
| `cable_routing` | Кабельные трассы: совместная прокладка, способы крепления, огнестойкость |
| `documentation` | Соответствие чертежей спецификации, оформление, маркировка |
| `normative_refs` | Устаревшие или неверные ссылки на нормативные документы |

Формат в JSON:
```
"category": "scs|cctv|access_control|intercom|fire_alarm|pa_system|tv_radio|dispatch|cable_routing|documentation|normative_refs"
```

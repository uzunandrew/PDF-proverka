### Структурированная кабельная система (СКС)

- Категория кабеля (Cat.5e / Cat.6 / Cat.6A) -- соответствие ТЗ и нормативам
- Коммутационное оборудование (патч-панели, кроссы, розетки)
- Серверная / кроссовая: площадь, кондиционирование, электропитание, заземление
- Количество портов на квартиру / этаж -- соответствие проекту
- Длина горизонтальных линий <= 90 м (по ГОСТ Р 53246)
- Магистральные линии (медь / оптика) -- тип и ёмкость
- Маркировка кабелей и портов -- наличие схемы маркировки
- Кабельные трассы: совместная прокладка со слаботочными/силовыми

### Видеонаблюдение (CCTV)

- Количество и типы камер -- соответствие плану размещения
- Зоны покрытия: входы, паркинг, дворовая территория, лифтовые холлы
- Разрешение камер и углы обзора -- достаточность для идентификации
- Видеорегистратор / сервер хранения: глубина архива (не менее 30 суток по СП 134)
- Питание камер: PoE / отдельное, наличие ИБП
- Кабельная инфраструктура (UTP / оптика / коаксиал)
- Мониторы оператора -- наличие поста охраны

### СКУД (система контроля и управления доступом)

- Точки доступа: входные группы, паркинг, технические помещения
- Типы считывателей (proximity, биометрия, ключ-карта)
- Контроллеры доступа: количество, ёмкость, резервирование
- Электромагнитные / электромеханические замки -- тип и характеристики
- Интеграция с домофоном и пожарной сигнализацией
- Разблокировка дверей при пожаре (интеграция с ОПС/СОУЭ)
- Кнопка экстренного открывания (Exit) -- наличие на путях эвакуации
- Питание и ИБП контроллеров

### Домофон / видеодомофон

- Вызывные панели: количество, расположение (главный вход, калитки)
- Абонентские мониторы / трубки -- тип, совместимость
- Связь с постом консьержа / охраны
- Электрозамок -- тип, мощность, совместимость с панелью
- Кабельная разводка (витая пара, коаксиал, 2-проводная)
- Количество абонентов на систему -- соответствие ёмкости

### ОПС / АУПС (пожарная сигнализация)

- Типы извещателей: дымовые, тепловые, ручные (ИПР) -- соответствие помещениям
- Размещение извещателей -- нормативные расстояния (СП 484.1311500.2020, табл. А.1)
- Адресные / безадресные шлейфы -- тип системы
- Приёмно-контрольные приборы (ПКП/ППКП) -- ёмкость, расположение
- Интеграция с СОУЭ: автоматический запуск оповещения
- Интеграция со СКУД: разблокировка дверей на путях эвакуации
- Интеграция с лифтами: опускание на 1-й этаж при пожаре
- Интеграция с вентиляцией: отключение общеобменной, включение противодымной
- Кабели шлейфов: огнестойкость (FRLS/FR), способ прокладки
- Резервное питание ПКП (АКБ не менее 24 ч дежурный + 1 ч тревога)

### СОУЭ (система оповещения и управления эвакуацией)

- Тип СОУЭ (1-5) -- соответствие этажности и назначению здания
- Зоны оповещения -- разделение по пожарным отсекам / секциям
- Речевые оповещатели: мощность, размещение, уровень звукового давления
- Световые оповещатели "Выход" -- на путях эвакуации
- Автоматический и ручной запуск оповещения
- Приоритет сигналов (пожарная тревога > прочие)
- Кабели: огнестойкость (FRLS/FR), прокладка в огнестойких коробах
- Резервное питание (АКБ)

### Телевидение и радио

- Антенное хозяйство: коллективная антенна / кабельное ТВ / оптика GPON
- Усилители сигнала: тип, количество, расположение
- Разводка по квартирам: коаксиал / оптика
- Головная станция / оптический узел -- расположение
- Количество ТВ-выводов на квартиру
- Радиофикация (3-я программа проводного вещания) -- при наличии требования

### Диспетчеризация

- Лифты: связь с диспетчерской, аварийные сигналы
- ИТП (индивидуальный тепловой пункт): датчики, сигналы аварий
- Пожарная автоматика: вывод сигналов на пульт ОДС / пожарную часть
- Водомерный узел: импульсные счётчики, передача данных
- Электроснабжение (АСКУЭ): счётчики с цифровым выходом, сервер сбора данных
- Каналы связи с диспетчерской (выделенная линия / GSM / Ethernet)
- Резервирование каналов связи

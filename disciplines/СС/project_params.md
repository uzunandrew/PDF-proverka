## Что искать в тексте (дисциплина СС)

- Нормативные ссылки -> проверить актуальность по нормативной базе дисциплины
- Тип и категория СКС (Cat.5e / Cat.6 / Cat.6A)
- Количество и типы камер видеонаблюдения
- Тип системы ОПС (адресная / безадресная / адресно-аналоговая)
- Тип СОУЭ (1-5)
- Способ приёма ТВ (коллективная антенна / кабельное / GPON)
- Состав систем диспетчеризации
- Противоречия между разделами проекта

## JSON-шаблон project_params

```json
{
  "scs_category": "Cat.5e",
  "cctv_cameras_count": 0,
  "access_points_count": 0,
  "intercom_panels_count": 0,
  "fire_alarm_type": "адресная",
  "soue_type": "2",
  "tv_system": "кабельное ТВ",
  "dispatch_systems": ["лифты", "ИТП", "пожарная", "водомерный"],
  "server_room_count": 1,
  "total_apartments": 0
}
```

## Описание полей

| Поле | Описание | Пример |
|---|---|---|
| scs_category | Категория кабеля СКС | "Cat.5e", "Cat.6", "Cat.6A" |
| cctv_cameras_count | Общее количество камер видеонаблюдения | 48 |
| access_points_count | Количество точек доступа СКУД | 12 |
| intercom_panels_count | Количество вызывных панелей домофона | 4 |
| fire_alarm_type | Тип системы пожарной сигнализации | "адресная", "безадресная", "адресно-аналоговая" |
| soue_type | Тип СОУЭ (1-5 по СП 3.13130.2009) | "2", "3", "4" |
| tv_system | Способ приёма телевидения | "коллективная антенна", "кабельное ТВ", "GPON" |
| dispatch_systems | Список систем, выводимых на диспетчерский пульт | ["лифты", "ИТП", "пожарная", "водомерный", "АСКУЭ"] |
| server_room_count | Количество серверных / кроссовых помещений | 1, 2 |
| total_apartments | Общее количество квартир в доме | 240 |

| sheet_type | sheet_type_label | Описание |
|---|---|---|
| `framing_plan` | Схема расположения элементов | План расположения колонн, балок, связей на отметке |
| `assembly_plan` | Монтажная схема | Монтажная схема с отправочными марками и порядком монтажа |
| `connection_detail` | Узел соединения | Детальный чертёж узла (сварного, болтового, фланцевого) |
| `cross_section` | Сечение элемента | Поперечное сечение балки, колонны, фермы с размерами |
| `general_view` | Общий вид конструкции | Общий вид каркаса, фермы, рамы с основными размерами |
| `truss_drawing` | Чертёж фермы | Геометрическая схема фермы, узлы, сечения стержней |
| `column_drawing` | Чертёж колонны | Колонна с базой, оголовком, рёбрами жёсткости |
| `beam_drawing` | Чертёж балки | Балка с рёбрами, отверстиями, опорными узлами |
| `bracing_scheme` | Схема связей | Вертикальные и горизонтальные связи, распорки |
| `specification` | Спецификация | Спецификация металлопроката и метизов |
| `title_block` | Титульный лист | Титульный лист, содержание |
| `general_notes` | Общие указания | Общие данные, указания по изготовлению и монтажу |
| `detail` | Деталь/узел | Мелкие детали, накладки, косынки, фасонки |
| `other` | Прочее | Не попадает в другие категории |

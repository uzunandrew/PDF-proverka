## Что искать в тексте (дисциплина ГП)
- Нормативные ссылки (СП 42.13330, СП 476.1325800, СП 82.13330, СП 4.13130, СП 59.13330 и др.)
- Площадь земельного участка, баланс территории
- Площадь застройки (здания, сооружения)
- Площадь озеленения (процент от участка)
- Количество парковочных мест (наземных, для МГН)
- Количество и типы площадок (детские, спортивные, хозяйственные, контейнерные)
- Ширина пожарных проездов, наличие разворотных площадок
- Перечень наружных инженерных сетей
- Тип водоотведения поверхностных вод (ливневая канализация, открытый водоотвод)
- Процент застройки участка

## Описание полей
| Поле | Описание | Пример |
|---|---|---|
| site_area_m2 | Площадь земельного участка, м² | 15000 |
| building_footprint_m2 | Площадь застройки (пятно здания), м² | 3200 |
| site_coverage_percent | Процент застройки участка | 21.3 |
| green_area_m2 | Площадь озеленения, м² | 4500 |
| green_area_percent | Процент озеленения от площади участка | 30 |
| hardscape_area_m2 | Площадь твёрдых покрытий (проезды, тротуары, площадки), м² | 5800 |
| parking_spaces_outdoor | Количество наземных парковочных мест | 45 |
| parking_spaces_mgn | Количество мест для МГН | 5 |
| playground_count | Количество площадок (все типы) | 4 |
| playground_types | Типы площадок | ["детская", "спортивная", "отдых", "хозяйственная", "контейнерная"] |
| fire_access_width_m | Ширина пожарного проезда, м | 3.5 |
| fire_access_sides | С скольких сторон здания обеспечен пожарный проезд | 2 |
| turnaround_present | Наличие разворотной площадки (при тупиковых проездах) | true |
| utility_networks | Перечень наружных инженерных сетей | ["В1", "К1", "К2", "Т1", "ЭС", "СС"] |
| drainage_type | Тип водоотведения поверхностных вод | "ливневая канализация" |
| terrain_slope_percent | Уклон территории, % | 1.5 |
| building_height_m | Высота здания, м (для определения требований к пожарным проездам) | 75.0 |
| resident_count | Расчётное количество жителей | 420 |

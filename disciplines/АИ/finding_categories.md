Категории замечаний для дисциплины АИ (Архитектурные решения — интерьер):

| category | Описание |
|---|---|
| `finishes` | Отделка помещений: выбор материалов, классы, соответствие назначению |
| `flooring` | Полы: типы покрытий, пороги, уклоны, гидроизоляция, звукоизоляция |
| `ceiling` | Потолки: высоты, конструкция подвесных/натяжных, встроенное оборудование |
| `walls` | Стены и развёртки: раскладка плитки, ниши, фартуки, отделка |
| `bathroom` | Санузлы: размещение приборов, отступы, гидроизоляция, вентиляция |
| `kitchen` | Кухни: рабочий треугольник, вытяжка, встроенная техника, розетки |
| `doors` | Двери: типоразмеры, направление открывания, фурнитура, звукоизоляция |
| `furniture` | Мебель: встроенная и корпусная мебель, размеры, привязки |
| `lighting` | Интерьерное освещение: встроенные светильники, подсветка, сценарии |
| `accessibility` | Доступность МГН: поручни, размеры, высоты, пороги |
| `documentation` | Соответствие чертежей спецификации, оформление, комплектность |
| `normative_refs` | Устаревшие или неверные ссылки на нормативные документы |

Формат в JSON:
```
"category": "finishes|flooring|ceiling|walls|bathroom|kitchen|doors|furniture|lighting|accessibility|documentation|normative_refs"
```

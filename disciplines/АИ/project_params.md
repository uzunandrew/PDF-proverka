## Что искать в тексте (дисциплина АИ)

- Типы отделки помещений (по ведомости отделки)
- Классы пожарной опасности отделочных материалов (КМ0–КМ5)
- Экологические классы материалов (Е1, Е0,5 — эмиссия формальдегида)
- Классы истираемости напольных покрытий (PEI I–V для керамогранита, AC1–AC5 для ламината)
- Ведомость отделки помещений (материалы полов, стен, потолков по помещениям)
- Ведомость дверных заполнений (типоразмеры, материал, фурнитура)
- Спецификация сантехнических приборов (марки, количество)
- Нормативные ссылки → проверить актуальность по нормативной базе дисциплины
- Противоречия между ведомостью отделки и чертежами

## JSON-шаблон project_params

```json
{
  "finish_class": "...",
  "floor_types": [],
  "wet_zone_finish": "...",
  "ceiling_type": "...",
  "ceiling_height_mm": 0,
  "door_types": [],
  "bathroom_fixture_brands": [],
  "kitchen_equipment": []
}
```

## Описание полей

| Поле | Описание | Пример |
|---|---|---|
| finish_class | Класс пожарной опасности отделки на путях эвакуации | "КМ1", "КМ2" |
| floor_types | Типы напольных покрытий в проекте | ["керамогранит 600×600", "ламинат AC4 33 класс", "линолеум КМ2"] |
| wet_zone_finish | Отделка мокрых зон (стены/пол) | "керамогранит, гидроизоляция обмазочная" |
| ceiling_type | Тип потолков (основной) | "натяжной", "подвесной Армстронг", "ГКЛ", "штукатурка" |
| ceiling_height_mm | Высота потолка в чистоте, мм | 2700 |
| door_types | Типы дверей в проекте | ["ДГ 21-9", "ДО 21-8", "ДПМ EI60"] |
| bathroom_fixture_brands | Марки сантехприборов | ["Roca", "Grohe", "Geberit"] |
| kitchen_equipment | Кухонное оборудование | ["варочная панель", "духовой шкаф", "посудомоечная машина", "вытяжка"] |

Категории замечаний для дисциплины ВК:

| category | Описание |
|---|---|
| `cold_water` | ХВС: ввод, разводка, стояки, поквартирное водоснабжение |
| `hot_water` | ГВС: приготовление, циркуляция, полотенцесушители, температурный режим |
| `sewerage` | Канализация: стояки, уклоны, ревизии, вентиляция стояков, гидрозатворы |
| `storm_drain` | Ливневая канализация: водоприёмные воронки, внутренние водостоки |
| `water_meter` | Водомерный узел: приборы учёта, фильтры, обратные клапаны, арматура |
| `pumps` | Насосные станции: повысительные, циркуляционные, дренажные, характеристики |
| `pipe_sizing` | Диаметры и расходы: гидравлический расчёт, потери напора, скорости |
| `insulation` | Изоляция: теплоизоляция ГВС, антиконденсатная изоляция ХВС |
| `materials` | Трубы и арматура: марки, материалы, соответствие ГОСТ |
| `documentation` | Соответствие чертежей спецификации, оформление, комплектность |
| `normative_refs` | Устаревшие или неверные ссылки на нормативные документы |

Формат в JSON:
```
"category": "cold_water|hot_water|sewerage|storm_drain|water_meter|pumps|pipe_sizing|insulation|materials|documentation|normative_refs"
```

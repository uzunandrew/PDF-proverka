## Что искать в тексте (дисциплина ВК)

- Нормативные ссылки → проверить актуальность по нормативной базе дисциплины
- Расчётные параметры (расходы воды, напоры, диаметры, уклоны)
- Марки труб и соответствие ГОСТ 32415-2013 (ПП), ГОСТ 18599-2001 (ПЭ)
- Характеристики насосного оборудования (Q, H)
- Параметры водомерного узла (калибр, потери напора)
- Температурные параметры ГВС (не менее 60°C)
- Противоречия между разделами ПЗ и чертежами

## JSON-шаблон project_params

```json
{
  "water_source": "...",
  "cold_water_pressure_bar": 0,
  "hot_water_system": "...",
  "daily_water_demand_m3": 0,
  "sewerage_type": "...",
  "storm_drain_type": "...",
  "pump_count": 0,
  "pipe_material_cold": "...",
  "pipe_material_hot": "...",
  "pipe_material_sewerage": "...",
  "water_meter_caliber_mm": 0
}
```

## Описание полей

| Поле | Описание | Пример |
|---|---|---|
| water_source | Источник водоснабжения | "городской водопровод", "скважина", "комбинированный" |
| cold_water_pressure_bar | Гарантированный напор в наружной сети ХВС, бар | 3.5 |
| hot_water_system | Система ГВС (приготовление) | "ИТП (пластинчатый теплообменник)", "ЦТП", "бойлер" |
| daily_water_demand_m3 | Суточное водопотребление здания, м³/сут | 120 |
| sewerage_type | Тип бытовой канализации | "самотёчная", "напорная", "комбинированная" |
| storm_drain_type | Тип ливневой канализации | "внутренние водостоки", "наружные водостоки", "комбинированная" |
| pump_count | Количество повысительных насосов (вкл. резерв) | 3 |
| pipe_material_cold | Материал труб ХВС | "ПП (полипропилен)", "ПЭ (полиэтилен)", "сталь оцинкованная" |
| pipe_material_hot | Материал труб ГВС | "ПП (полипропилен)", "медь", "сталь оцинкованная" |
| pipe_material_sewerage | Материал труб канализации | "ПП (полипропилен)", "ПВХ", "чугун" |
| water_meter_caliber_mm | Калибр водомера на вводе, мм | 50 |
